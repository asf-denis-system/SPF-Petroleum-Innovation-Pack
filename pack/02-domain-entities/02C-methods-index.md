# Methods Index: Upstream Petroleum Development

Навигационный индекс всех методов (методик, процессов) управления пластовой системой.

---

## Что такое метод?

Per FPF: **Метод** — это повторяемый способ сделать что-то, который производит предсказуемый результат. Методы выполняются ролями и производят рабочие продукты.

Метод — это НЕ:
- Инструмент (инструмент поддерживает метод)
- Пошаговая инструкция
- Рабочий продукт (это результат метода)

[Innovation Project Template](../04-work-products/02-innovation-project-template.md)

## Примечания

- Каждый метод описывается подробно в папке 03-methods/
- Методы часто используются вместе (например, M.001 + M.002 + M.003 для формирования плана разработки)
- Методы верификации (UP.M.010) проводятся регулярно для обновления статуса принципов
- Все 13 методов согласованы с концепцией технологических состояний продукции в цепочке
