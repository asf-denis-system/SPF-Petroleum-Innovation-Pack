# [UP.M.004] Анализ энергетического баланса пластовой системы

## Определение

**Метод** — расчёт энергии, изначально накопленной в пласте (избыточное давление, энергия флюидов), её расходования при отборе нефти и восполнения при поддержке, прогнозирование достаточности энергии для разработки месторождения.

**Простая идея:** Пласт — это батарея. У неё есть энергия (давление). Каждый барель отобранной нефти «разряжает» батарею. Без подзарядки (ППД) батарея разрядится и месторождение упадёт в добыче.

## Цель

1. **Количественно оценить** запас энергии в пласте
2. **Рассчитать**, как быстро она расходуется при разных темпах отбора
3. **Спрогнозировать** нужна ли ППД и когда её запускать
4. **Оптимизировать** темп отбора (не сжигать энергию впустую)

## Входные данные

| Входные данные | Описание |
|---|---|
| **Геологические запасы** | Объём нефти в пласте (млн т) |
| **Начальное давление** | Давление на глубине центра пласта (МПа) |
| **Вязкость** | Динамическая вязкость нефти (мПа·с) |
| **Проницаемость** | Абсолютная проницаемость пласта (мД) |
| **Сжимаемость** | Сжимаемость флюидов и поры (1/МПа) |
| **Плотность нефти** | ρ (кг/м³) |
| **Скв. программа** | Кол-во, размещение, дебиты скважин |

## Процесс

### Этап 1: Расчёт начальной энергии (1 неделя)

Пласт имеет несколько источников энергии:

**1. Аквифер (напор воды)**
- Если под пластом есть аквифер → вода подпирает нефть
- Энергия ≈ давление столба воды (50–100 м воды = 0.5–1 МПа)

**2. Газовая шапка (расширение газа)**
- Если есть газовая шапка → при падении давления газ расширяется
- Энергия ≈ упругость газовой подушки (часто 0.2–0.5 начального давления)

**3. Упругостные сил (elastic energy)**
- Недиссоциированная сжимаемость поры, нефти, воды
- Энергия: $E = V \cdot c \cdot \Delta P$, где V = объём пласта, c = сжимаемость, ΔP = перепад давления

### Расчёт:
$$\text{Начальная энергия} = E_{\text{аквифер}} + E_{\text{газовая шапка}} + E_{\text{упругость}}$$

### Этап 2: Прогноз давления без ППД (1–2 недели)

Используя гидродинамическую модель (или аналитические уравнения):

$$P(t) = P_0 - \frac{q \cdot t}{c \cdot V}$$

где q = дебит, t = время, c = сжимаемость, V = объём.

Для каждого темпа отбора (консервативный, базовый, агрессивный) рассчитать:
- **Темп падения давления** (МПа/год)
- **Время достижения минимально приемлемого давления** (Pmin — ниже этого забойное давление не достаточно для поднятия нефти)
- **КИН** при этом режиме

**Результат:** график давления vs время.

### Этап 3: Расчёт потребной ППД (1 неделя)

Если без ППД давление упадёт ниже Pmin к году 10 → нужна ППД.

**Расчёт мощности ППД:**
- Какой объём воды нагнетать, чтобы давление в пласте оставалось выше Pmin?
- На какое давление нагнетать скважине?

$$q_{\text{ППД}} = \frac{dc}{dt} \cdot V$$

где dc/dt = скорость падения давления без поддержки; V = объём.

**Результат:** разработать график: "ППД включить в год X, объём нагнетания Y м³/день, давление нагнетания Z МПа".

### Этап 4: Экономический анализ (1 неделя)

Для каждого режима (с ППД, без ППД, отложенная ППД) рассчитать:
- **Дополнительная добыча** привет ППД (млн т)
- **Стоимость работ ППД** (буровые, оборудование, операционные расходы)
- **NPV** с ППД vs без ППД

**Уравнение:**
$$\text{NPV}_{\text{с ППД}} = \sum_t \frac{\text{Доход}_t - \text{Затраты}_t}{(1+r)^t}$$

## Пример

**Месторождение Северное, пласт Нь1**

- Геол. запасы: 100 млн т
- Начальное давление: 45 МПа
- Есть слабый аквифер и газовая шапка
- Система 8 скважин, начальный дебит 50 тыс. т/год

**Расчёты:**
- Без ППД: давление упадёт 45 → 20 МПа к году 15 → поле заканчивается
  - Добыча: 750 М т (75% от геол. запасов)
  - КИН = 75%
  - NPV = $450 млн

- С ППД (запуск год 8): давление поддерживается ≥ 25 МПа
  - ППД объём 5 млн м³ воды/год, давление 60 МПа
  - Добыча: 850 М т (85% от геол. запасов)
  - КИН = 85%
  - Затраты ППД: $50 млн
  - NPV = $480 млн (выигрыш +$30 млн)

**Вывод:** ППД окупает себя, запускаем в год 8.

## Когда этот метод не работает

- **Мало данных о пласте** → неточность в расчётах
- **Двухфазный режим** (газ и нефть вместе) → вязкость меняется, модель усложняется
- **Аномальное геохимическое давление** → стандартные формулы не применимы
