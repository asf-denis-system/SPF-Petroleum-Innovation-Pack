# [UP.M.001] Анализ гетерогенности пласта

## Определение

**Метод** — систематическое выявление, картирование и количественная оценка пространственных вариаций в геологических, петрофизических и геометрических свойствах пласта на основе:
- Скважинных данных (керн, каротаж)
- Сейсмической информации
- Структурного анализа
- Геологического моделирования

## Цель

1. **Идентифицировать** зоны различной проницаемости, пористости, структурных особенностей
2. **Картировать** их пространственное распределение
3. **Количественно оценить** масштаб гетерогенности
4. **Прогнозировать** поведение флюидов в неоднородной среде
5. **Избежать ошибок** (тип FM.PHYS.002, FM.PHYS.003)

## Входные данные

| Входные данные | Описание | Источник |
|---|---|---|
| Каротажные данные | ГК, НК, АК, ЭЛК, ПС (≥ 3 скважины) | Скважины |
| Керновые образцы | ≥ 50 м керна с анализом пористости, проницаемости, минерального состава | Объект-скважина |
| Сейсмические данные | 3D сейсмика (если есть) или 2D профили | После обработки |
| Структурная информация | Карты высот подошвы, кровли, толщины пласта; трещины | Предыдущие интерпретации |
| Аналогичные месторождения | Описание гетерогенности в подобных геолого-тектонических условиях | Публикации, отчёты |

## Процесс (шаги)

### Этап 1: Сбор и анализ каротажных данных (1–2 недели)
1. Проверить качество и калибровку каротажа
2. Для каждой скважины вычислить:
   - Пористость (из АК/НК)
   - Проницаемость (кросс-плот, эмпирические зависимости)
   - Литология (ГК, сопротивление)
   - Флюидные контакты (АК, сопротивление)
3. Построить колонки всех скважин рядом (correlation) — видны вариации слойности

### Этап 2: Интерпретация керна (1–2 недели)
1. Описание образцов микроскопически и макроскопически
2. Выделить фации (изменения в составе, структуре, цвете)
3. Связать фации с каротажом: "эта фация на кривых выглядит так"
4. Määritellä связь поровых размеров и распределений с проницаемостью

### Этап 3: Видовая дифференциация (1 неделя)
1. На основе фаций выделить **зоны**:
   - Высокопроницаемые пески (гидрокси-слпицы, песчаники с видимой пористостью)
   - Низкопроницаемые зоны (глины, карбонаты с микрокавернозностью)
   - Переходные зоны (смешанные отложения)
2. Для каждой зоны вычислить:
   - Ожидаемая пористость (%)
   - Ожидаемая проницаемость (мД)
   - Объёмная доля в пласте (%)

### Этап 4: Картирование структуры и трещин (1–2 недели)
1. Использовать сейсмику для:
   - Выявления подземных разломов
   - Изменений толщины пласта
   - Экспортации подошвы и кровли
2. Интерпретировать естественные трещины:
   - Диастрофизм (стресс-ориентированные трещины)
   - Признаки в каротаже (микро-разломы)
3. Построить структурную карту с отмеченными негативными особенностями

### Этап 5: 3D моделирование (2–3 недели)
1. Инициировать сетку (grid) 3D модели на основе структуры
2. Апрiory распределить свойства (пористость, проницаемость) между ячейками:
   - В скважинах: точные значения из каротажа
   - Между скважинами: интерполяция с учётом фаций
3. Использовать variogram-анализ для определения масштаба автокорреляции: "на каком расстоянии гетерогенность становится заметной?"
4. Калибровать модель:
   - Может ли она объяснить разброс дебитов скважин?
   - Соответствует ли она сейсмике?

### Этап 6: Гидравлическая связность (1 неделя)
1. На основе моделей проницаемости определить:
   - Какие области имеют гидравлическую связь (могут обмениваться давлением)?
   - Какие области изолированы барьерами?
2. Это важно для ППД и планирования скважин

### Этап 7: Подготовка портфеля моделей (1 неделя)
1. Нужна не одна, а несколько моделей гетерогенности:
   - **Пессимистичная** (высокая гетерогенность → быстрая неоднородность потоков)
   - **Базовая** (среднее понимание)
   - **Оптимистичная** (более однородный пласт)
2. Каждая модель должна быть геологически обоснована

## Пример

**Месторождение A (Западная Сибирь), пласт Ю1**

- 5 скважин, 250 м керна, 3D сейсмика есть
- Каротаж показывает:
  - Скважина№1: средняя пористость 20%, проницаемость 150 мД
  - Скважина№2: низкая пористость 8%, проницаемость 5 мД (в 30 раз меньше!)
  - Остальные 3 — промежуточные
- Вывод: гетерогенность высокая, разброс в 30 раз
- Анализ керна показал: Скважина№1 — скважинный песок (хорошо рассортирован), Скважина№2 — скважинный ил с глинистым заполнением (низкая проницаемость)
- Сейсмика подтверждает: На юге (Скв.№2) мощность пласта меньше и амплитуда слабее → изменение литологии

**Результат:** Модель показала, что ППД-скважину нельзя размещать рядом с Скв.№2 (не получит поддержку давления) — смещение на 500 м к северу (к Скв.№1).

## Когда этот метод не работает

- **Недостаточно скважин** (< 2 скважин) → малая информация о гетерогенности
- **Нет керна** → только каротаж, зависим от эмпирических зависимостей, ошибки в разы
- **Трещины в подземе** → нужны тонко специальные методы (микросейсмика, давлениетесты)
- **Динамическое поведение не совпадает с моделью** → модель неполная, есть механизмы, которые не учтены
